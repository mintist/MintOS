宏定义说明

#define TASK_SCHED_ARRAY   1

采用数组作为任务就续表，支持无限个数的任务

#define TASK_SCHED_UCOS128   0

ucosii查表法，最大支持128个任务

#define TASK_SCHED_UCOS256   0

ucosii查表法，最大支持256个任务

#define TASK_SCHED_UCOS512   0

ucosii查表法，最大支持512个任务

#define TASK_SCHED_UCOS64   0

ucosii查表法，最大支持64个任务

#define USER_TASKS   200

用户定义自己需要用的任务数目

void m_set_prio ( uint32_t  prio)

使能任务就绪表

参数
prio,:依据你所选择的任务调度算法指定任务优先级 可选的任务调度算法:
  • TASK_SCHED_ARRAY 1~任意，只要你内存够大
  • TASK_SCHED_UCOS64 1~62
  • TASK_SCHED_UCOS128 1~126
  • TASK_SCHED_UCOS256 1~254
  • TASK_SCHED_UCOS512 1~510
返回值
无
