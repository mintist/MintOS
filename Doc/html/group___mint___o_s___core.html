void m_sched ( void  )

系统最核心的函数之一：任务调度函数，总是去调度就绪 任务表中优先级最高的任务。简要步骤如下： 

1. 寻找就续表中的最高优先级
    2. 如果比当前优先级更高，则
        2.1 设置最该优先级以及当前优先级栈地址
        2.2 切换任务
参数
无
返回值
无
void m_start ( void  )

系统初始启动函数，一般位于TaskStart（任务启动任务）被 创建之后调用，简要步骤如下： 

1. 获取最高优先级任务
2. 设置最该优先级以及当前优先级栈地址
3. 调用m_start_high启动Mint OS
参数
无
返回值
无
